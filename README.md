# 🎓 ConsoleApp5 — Student Data Entry System

This project is a C# Console Application that stores student details in a SQL Server database using ADO.NET.

It accepts input from the user and inserts the data into the database securely using parameterized queries.

---

## 🧰 Technologies Used

- C#
- .NET Framework Console Application
- SQL Server
- ADO.NET
- Visual Studio

---

## ✨ Features

✔ Accepts student details from user input  
✔ Connects to SQL Server database  
✔ Inserts records into the Student table  
✔ Uses parameterized queries to prevent SQL injection  
✔ Simple and beginner-friendly implementation  

---

## 📥 Input Fields

The application collects the following information:

- Student Number  
- Student Name  
- Student Address  
- Student Marks  

---

## 🗄️ Database Requirement

Create a database named **demo** and a table named **Student** with appropriate columns:

Example structure:

- StuNo (INT)  
- StuName (VARCHAR)  
- StuAddress (VARCHAR)  
- StuMarks (INT)  

---

## ▶️ How to Run

1. Open the project in Visual Studio  
2. Ensure SQL Server is running  
3. Update the connection string if needed  
4. Run the application  
5. Enter student details when prompted  

---

## 🎯 Learning Outcome

This project demonstrates:

- Basic ADO.NET database connectivity  
- SQL INSERT operation from C#  
- Handling user input in console applications
